### 关键发现

1. **身份锚定缺陷**（#3）是最严重的问题——需要区分 @自己 vs @别人
2. **话题感知缺失**（#2）导致回复浮在表面——需要给 agent 群聊话题摘要
3. **"来群里唠嗑"**这类语义偏差——需要在 prompt 中强调 agent 是在「当前群」聊天
4. 表情包评价（#1, #5）表现自然，说明 LLM 基础对话能力足够
