# 架构
- 消息收发器
  - 接收器 
  - 上下文追踪器 ctx_tracker
  - 用户名 / UID 管理 user_cache 
  - 历史消息记录 history
  - 回复 sender
    - 消息切分
- 表情包管理器（提供工具调用：收藏、列出）
- 冲浪模拟器（网页渲染解析、链接访问）
- Linux Sandbox 模型可完整控制一个 Docker 容器供其操作
- 消息 Peek
- 闲时处理器（冲浪、整理表情包）
- 日志管理器
  - 模型日志管理器（存取模型响应以供审阅）
  - 运行时日志管理器（存取程序运行相关日志）
- WebUI
  - 通用配置管理
  - 日志管理
